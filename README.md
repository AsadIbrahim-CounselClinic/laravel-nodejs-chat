## About CHAT-APP
SIMPLE CHAT APP USING LARAVEL, MYSQL AND SOCKET IO

### MAIN APP SERVER
- SETUP LARAVEL .env
- SETUP DB DETAILS
- RUN `npm install`
- RUN `php artisan migrate`
- RUN `composer install`
- RUN `php artisan serve`
- RUN `npm run dev`

### CHAT SERVER
- SETUP /chat-server .env (with DB details only)
- RUN `npm install`
- RUN `npm run dev`
